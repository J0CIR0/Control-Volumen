Control de Volumen por Gestos de Mano

Este programa utiliza la visión por computadora y el procesamiento de audio
para controlar el volumen del sistema a través de gestos de mano.
Utiliza la biblioteca MediaPipe para el seguimiento de manos y OpenCV para el procesamiento de imágenes.

Requisitos:
Python 3.x
OpenCV
NumPy
MediaPipe
PyCaw

Puedes instalar las dependencias necesarias ejecutando el siguiente comando:
pip install opencv-python mediapipe pycaw

Uso:
Ejecuta el programa en un entorno Python compatible.
Coloca tu mano frente a la cámara.
Mueve tu mano hacia arriba o hacia abajo para ajustar el volumen.
El volumen se ajustará en tiempo real según el gesto de tu mano.
